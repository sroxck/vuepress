# 目标

模仿vue,手写一个属于自己的mvvm框架

# 流程分析

视图 -> 编译 -> 渲染

数据 -> 劫持 -> 编译

# 流程图

$el视图经过complier编译,进行render渲染,通过watcher观察数据变化,通知ovserver劫持
$data 数据经过observer劫持,通过dep依赖收集器触发观察者,观察者触发更新

data进行数据代理,函数代理 observer递归进行数据劫持,并拿到名单通知watch进行视图更新 dp存储watch,循环所有watcher 调用更新方法 watcher返回表达式,并执行回调函数进行编译渲染 通过watcher的回调,把设置好的表达式的值,赋值给对应的节点 编译主要做了以下事情,都是通过watcher的回调把表达式的值赋值给节点,需要注意都是 使用Dep.target静态变量存储watcher的实例,然后通过dep名单循环去通知更新
