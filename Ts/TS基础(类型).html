# title: TS基础-类型 tags: ts

# TS基础

为了让程序有价值，我们需要能够处理最简单的数据单元：数字，字符串，结构体，布尔值等。 TypeScript支持与JavaScript几乎相同的数据类型，此外还提供了实用的枚举类型方便我们使用

# 布尔值

let isBool: boolean = false

# 字符串

let isString: string = "字符串"

# 数字

let isNumber: number = 100

# 数组

普通形式的数组:let isArray: number[] = [1,2,3]
泛型形式的数组:let isArray: Array<number> = [1,2,3]

# 对象

普通对象:let isObj1:{} = {any:333}
限制属性名和类型:let isObj2:{size:string} = {size:'22'}

# 任意类型

TIP

使用any类型后,值可以为任意类型,在不知道具体类型的时候可以使用,尽量不使用any类型

使用any类型,值可以为数字或者字符串或者其他任意类型
let isAny: any = "33"
let isAnyTwo: any = 333
数组使用any类型后,数组中每一项元素值可为任意类型
let isAnyArray:any[] = [1,'1',false]

# 元组

元组最重要的特性是可以限制数组元素的个数和类型，它特别适合用来实现多值返回
元组,表示一个已知类型和数量的数组,如下例:isTuple数组只能有2个元素,类型是String,Number,类型顺序也必须一致
let isTuple : [string, number] = ['2',3]

# 枚举

数字类型枚举可以手动设置起始索引,后面的值自动++,如果字符串枚举和数字枚举混用,所有元素都需要手动设置
enum Color {red = 1,pink,black}
let c: Color = Color.pink // c = 2

# 空值

空值 void,函数如果没有返回值可以设置,变量设置没有意义,因为赋值只能设置undefined和null
function returnVoid():void{}

# 联合类型

可以为一个变量指定多个类型,类型于或的关系
let lhlx: number|null = null

# 函数返回值类型设置

函数返回值类型格式: 写在参数列表括号之后
function returnString():string{return "string"}
let isNull: null= null

# never类型

返回never类型的函数必须有永远无法到达的终点,如异常

function returnNever():never{
    throw new Error() // 被异常中断
}

# 类型断言

当你知道类型是什么的时候,通常这会发生在你清楚地知道一个实体具有比它现有类型更确切的类型。类型是any 但是我知道他实际是String
let stringLength: any = "hello";
类似是制类型转换(强制转换需要先转换到特定格式,后面有讲),有以下两种格式
let stringLengthNuber1: number = (<string>stringLength).length
let stringLengthNuber2: number = (stringLength as string).length
